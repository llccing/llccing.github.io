# OpenClaw Backup - Tencent Cloud Lighthouse

This repository contains a complete backup of OpenClaw configuration and memory from a Tencent Cloud Lighthouse instance.

## Contents

- `openclaw.json` - Main configuration file
- `workspace/` - Personal workspace with memory files, identity, and user context
- `extensions/` - Installed extensions (Feishu, WeChat, QQBot, etc.)
- `agents/` - Agent configurations
- `cron/` - Scheduled tasks
- `memory/` - Session memory storage
- Other runtime directories
