# 《至尊宝在人间》第二季 — 角色圣经 & 世界观设定

## 一、故事概述

**标题：** 《至尊宝在人间：九九八十一难》
**时间线：** 2027年3月 — 2029年底（约3年）
**地点：** 主要在哈尔滨，部分章节涉及上海、深圳
**接续：** 紧接原作第10章《一年后，地铁口我遇见了另一个她》之后

至尊宝（刘振宝）回到哈尔滨一年，以为人生安定了——前台国企，一万八月薪，准时下班，周末陪父母。但新的"九九八十一难"才刚刚开始。这不是一个逆袭的故事，而是一个普通人反复被生活锤、反复站起来的故事。

## 二、核心人物

### 主角：至尊宝 / 刘振宝
- **年龄：** 35岁（2027年）
- **身份：** 哈尔滨某国企前端开发 → 团队负责人
- **性格：** 外表随和内心倔强，嘴上认命心里不甘，自嘲是保护色
- **核心矛盾：** 在"安稳"和"不甘心"之间反复拉扯
- **口头禅：** "都差不多，都是打工的"
- **习惯：** 凉了的咖啡也喝、地铁里闭眼假装睡觉、压力大时算房贷
- **前史要点：**
  - 上海P6，月薪2.8万，被裁
  - 和紫霞纠缠三年，最终她嫁给了上海本地人
  - 父亲住院后决定回哈尔滨
  - 回来一年，薪资1.8万，带小团队

### 紫霞 / 林紫霞
- **年龄：** 33岁
- **身份：** 深圳某公司产品经理 → 后续变化
- **性格：** 独立、骄傲、嘴硬心软
- **与至尊宝的关系：** 前任，认识十年，彼此最懂但始终错过
- **出场节奏：** 前30章不出场（只在回忆中），第七劫（约61-70章）回归
- **关键设定：** 她的婚姻并不幸福，但她不是"等着被拯救的女人"

### 林晚
- **年龄：** 30岁
- **身份：** 哈尔滨某医院护士
- **性格：** 务实、温暖、有点毒舌、笑起来眼睛弯弯的
- **与紫霞的区别：** 紫霞是月光——美但抓不住；林晚是灶火——不耀眼但暖
- **出场：** 第四劫（约31-40章）通过相亲认识
- **核心弧线：** 她不是紫霞的替代品，她有自己的伤疤和故事（丧偶，带一个5岁女儿）
- **与至尊宝的关系演变：** 尴尬相亲 → 慢慢靠近 → 因紫霞回归而危机 → 最终的选择

### 老赵 / 赵大壮（猪八戒型）
- **年龄：** 38岁
- **身份：** 同事，后端开发，公司老员工
- **性格：** 贪吃、话多、好色但有底线、关键时刻讲义气
- **口头禅：** "刘哥，中午吃啥？" / "这事儿包在我身上"（然后搞砸）
- **家庭：** 老婆管得严，怕老婆但嘴上不承认
- **作用：** 喜剧担当 + 至尊宝的镜像（老赵选择了安稳，至尊宝在纠结要不要选）

### 小陈 / 陈默（沙僧型）
- **年龄：** 27岁
- **身份：** 同事，前端新人，至尊宝带的第一个徒弟
- **性格：** 话少、做事靠谱、偶尔冒出一句话直击要害
- **背景：** 农村出来的孩子，家里条件不好，特别珍惜工作机会
- **作用：** 至尊宝看到年轻时的自己 + 关键时刻最可靠的人

### 菩提大爷 / 孙德福
- **年龄：** 68岁
- **身份：** 菩提便利店老板，退休工人
- **性格：** 看似絮叨实则通透，说话像打禅机但其实就是活明白了
- **背景：** 年轻时是国企工人，下岗后开便利店30年，老伴去世，儿子在国外
- **作用：** 精神导师，但不是全知全能的——他也有自己的遗憾和软弱
- **出场节奏：** 穿插出现，每个重要节点提供一次点拨

### 王海（牛魔王型）
- **年龄：** 36岁
- **身份：** 至尊宝在上海的前同事/前室友
- **性格：** 聪明、有野心、能说会道、亦正亦邪
- **背景：** 也被裁了，但没有认命，转头搞创业
- **与至尊宝的关系：** 既是最好的朋友，也是最危险的诱惑
- **核心事件：** 第五劫拉至尊宝回上海创业，第六劫创业出事

### 母亲 / 刘玉兰
- **年龄：** 62岁
- **性格：** 嘴碎、好面子、但对儿子的爱是沉默的行动派
- **核心弧线：** 不只是"等儿子回家的妈妈"，她有自己的社交圈、自己的骄傲和委屈
- **关键情节：** 催婚（但方式很真实）、父亲病重时她的坚强与崩溃

### 父亲 / 刘建国
- **年龄：** 64岁
- **身份：** 退休工人，有心脏病
- **性格：** 沉默寡言，不善表达，但偶尔一句话能让至尊宝哭出来
- **关键线索：** 父亲年轻时也曾"闯关东"去外地打工，也经历过回家的挣扎——历史在重复

## 三、世界观设定

### 大话西游 → 西游记 → 脱离西游 的隐喻进化

**第一阶段（1-30章）：大话西游的回响**
- 延续原作风格，至尊宝还在用大话西游的经典台词自嘲
- 但频率降低，每3-5章出现一次
- 暗示：他还没有真正走出那个"角色"

**第二阶段（31-70章）：西游记原著的深层映射**
- 隐喻转向西游记原著——心猿意马、八十一难、真假美猴王
- 至尊宝开始思考：戴上金箍是成长还是束缚？
- 菩提大爷成为"须菩提祖师"的现代版

**第三阶段（71-100章）：脱离西游**
- 至尊宝不再需要用孙悟空来定义自己
- 他就是刘振宝，一个普通人
- 最后一章如果还出现西游的梗，是带着微笑的告别

### 哈尔滨作为"取经路"
- 松花江 = 流沙河（平静之下暗流涌动）
- 中央大街 = 长安大道（繁华背后的孤独）
- 冬天的暴雪 = 火焰山（极端环境下的考验）
- 菩提便利店 = 灵台方寸山（获得智慧的地方）

### 时代背景（2027-2029）
- AI 冲击传统开发岗位
- 35岁程序员的出路焦虑
- 东北振兴的希望与现实
- 小城市的人情温暖与窒息感并存

## 四、叙事规则

1. **每章2500-3000字**（中文字符）
2. **日记体为主**，但穿插：微信聊天记录、第三人称旁白、梦境/闪回
3. **每章结尾必须有钩子**——要么悬念、要么情感冲击、要么反转
4. **感官描写不能少**——气味、温度、声音、触感，这是原作的灵魂
5. **不美化苦难**——穷就是穷，难就是难，不要鸡汤化
6. **幽默是盔甲**——至尊宝越痛苦越自嘲，这是他的人设
7. **不写圣人**——每个角色都有自私、软弱、矛盾的一面
8. **方言适度**——东北话点缀但不过度，保持可读性
